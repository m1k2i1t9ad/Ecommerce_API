# eCommerce API

## Overview

The eCommerce API provides a robust backend solution for managing an online store. It allows users to perform operations such as product management, order processing, and user authentication. This API is built with Django and is designed to be scalable and secure.

## Features

- **User Authentication**: Secure user registration and login with JWT tokens.
- **Product Management**: Create, read, update, and delete products.
- **Order Processing**: Manage customer orders, including order creation and status updates.
- **Shopping Cart**: Support for adding and removing items from a shopping cart.
- **Search Functionality**: Search products by name, category, or price range.
